# Jarvis
Jarvis is a simple AI-powered virtual assistant built with Python. It can take voice commands, search the web, open applications, fetch information, and automate daily tasks. The project uses speech recognition, text-to-speech, and custom logic to create a smart, customizable personal assistant
 Features
 Voice Interaction

Understands your voice commands

Responds using text-to-speech (TTS)

 Internet-Based Tasks

Google search

YouTube search

Tells weather & time

Gives Wikipedia summaries

Reads latest news

 System Control

Opens apps (Chrome, VS Code, Notepad, etc.)

Plays music

Opens files & folders

System commands (shutdown, restart, etc.)

 Productivity Utilities

Sends emails (optional setup)

Creates notes & reminders

Performs calculations

Tells jokes, fun facts, quotes

 Tech Stack

Python 3

SpeechRecognition – For voice input

pyttsx3 / gTTS – For voice output

Wikipedia API

Custom Python modules
